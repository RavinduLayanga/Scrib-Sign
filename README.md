# Scrib-Sign

**Auto Caption Generation for Sign Language Videos**

## Overview

Scrib-Sign is a web-based application developed as a **second-year university group project** to explore automatic caption generation for sign language videos using **machine learning and computer vision** techniques.

The system allows users to upload sign language videos or video links and generates text captions to improve accessibility for deaf and hard-of-hearing communities.

---

## Features

* Upload sign language videos or YouTube links
* Automatic video frame extraction
* Gesture analysis using machine learning
* Auto-generated text captions
* Editable captions
* Fully implemented frontend and backend
